# so
Linux远程登录脚本
适用于堡垒机登录操作别的linux机器

# 界面
```python
[zhu@so]$ sh so.sh
#####################
    SSH Login Platform
#####################

序号 |       主机      | 说明
-----------------------------------------
  1  |       127.0.0.1 | 主机说明
-----------------------------------------
```
选择相应的序号，登录到对应的主机
